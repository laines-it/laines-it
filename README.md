## About me

I like projects where algorithms meet real interaction: board-game search, multiplayer protocols, tournament systems, simulators, and low-level experiments. Recently my work has centered around Xiangqi: engine research, Go services, and a browser multiplayer client.

I am comfortable moving between layers: from C/C++ and Go internals to TypeScript/React interfaces, with Python as a fast tool for experiments and analysis.

## Current focus

- Xiangqi engines and infrastructure: search, game logic, UCCI-related tooling, and web-facing services.
- Multiplayer web clients: React, Vite, TypeScript, WebSocket transport, and protobuf contracts.
- Algorithmic and systems practice: simulators, neural-network experiments, language analyzers, assembly homework, and functional programming exercises.
